Контрольні запитання:

  1. Що таке CSS і яка його роль у веб-розробці? CSS (Cascading Style Sheets) – це мова стилів, що використовується для задання вигляду документів, написаних мовою розмітки HTML. CSS дозволяє розділити зміст (HTML) від представлення (CSS), що робить веб-сторінки більш структурованими та легше керованими.
  2. Які основні властивості в CSS і як вони задаються? Основні властивості в CSS включають:
    • color: Задає колір тексту.
    • background-color: Задає фоновий колір елемента.
    • font-size: Задає розмір шрифту.
    • margin: Задає зовнішні відступи.
    • padding: Задає внутрішні відступи.
  3. Що таке селектори в CSS? Наведіть приклади різних типів селекторів. Селектори в CSS використовуються для вибору елементів на веб-сторінці, до яких будуть застосовані стилі. Приклади різних типів селекторів:
    • Селектори за тегом: css p { color: blue; }
    • Класові селектори: css .myClass { font-size: 20px; }
    • Ідентифікаторні селектори: css #myId { background-color: yellow; }
    • Класові селектори використовуються для стилізації групи елементів і позначаються через крапку (.). Один клас може бути застосований до багатьох елементів.
    • Ідентифікаторні селектори використовуються для стилізації одного унікального елемента і позначаються через решітку (#). Один ідентифікатор не повинен повторюватися у межах документа.
  5. Ідентифікаторні селектори використовуються для стилізації одного унікального елемента і позначаються через решітку (#). Один ідентифікатор не повинен повторюватися у межах документа.
  6. Як записати коментарі в CSS? Коментарі в CSS записуються наступним чином: /* Це коментар */
  7. Як змінити фоновий колір, колір тексту та розмір шрифту для елементу? css .myElement { background-color: lightblue; /* Фоновий колір */ color: black; /* Колір тексту */ font-size: 18px; /* Розмір шрифту */ }
  8. Як задати межі (borders), відступи (margins) та внутрішні відступи (padding) для елементів в CSS? css .myElement { border: 2px solid black; /* Межі */ margin: 20px; /* Зовнішні відступи */ padding: 15px; /* Внутрішні відступи */ }
